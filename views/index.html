API de Gestión de Películas

Bienvenido a la API RESTful para gestionar películas y directores.

Endpoints Disponibles:

  • /api/peliculas
  • /api/directores

Puede utilizar Postman para probar las funcionalidades de POST, PUT y DELETE.
