API de Gestión de Películas

Una API para administrar una colección de películas y directores.

Acerca de esta API

Este proyecto es una API RESTful completamente funcional construida con Node.js, Express y MongoDB. Permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre recursos de películas y directores.

Características Principales

  • Gestión (CRUD): Soporte para crear, leer, actualizar y eliminar películas y directores.
  • Búsqueda: Búsqueda de películas por título y listado de películas por director.
  • Consultas: Paginación y ordenamiento en el listado de películas.
  • Estructura: Desarrollada para una separación de responsabilidades.

Visión a Futuro

En futuras versiones, planeo expandir la funcionalidad para incluir nuevos recursos como Series, Actores y reseñas de usuarios, convirtiendola en una solución para la gestión de contenido en general.

Endpoints de Directores

GET /api/directores

Obtiene una lista de todos los directores.

GET /api/directores/:id/peliculas

Obtiene una lista de todas las películas de un director específico. Reemplazar :id con el ID del director.

POST /api/directores

Crea un nuevo director. Se debe enviar el siguiente JSON en el body:

{ "nombre": "Christopher Nolan", "nacionalidad": "Británico" }
DELETE /api/directores/:id

Elimina un director específico por su ID. Solo funciona si el director no tiene películas asociadas.

Endpoints de Películas

GET /api/peliculas

Obtiene una lista de todas las películas. Soporta paginación con page y limit, y ordenamiento con sortBy (ej: sortBy=titulo_asc).

GET /api/peliculas/:id

Obtiene una película específica por su ID.

GET /api/peliculas/buscar/:termino

Busca películas cuyo título coincida parcialmente con el término de búsqueda. Reemplazar :termino con la palabra a buscar.

POST /api/peliculas

Crea una nueva película. Se debe enviar el siguiente JSON en el body de la petición:

{ "titulo": "Interestelar", "genero": "Ciencia Ficción", "añoEstreno": 2014, "director": "ID_DEL_DIRECTOR_CREADO_PREVIAMENTE", "duracion": 169, "calificacion": 10 }
PUT /api/peliculas/:id

Actualiza una película existente. Se pueden enviar uno o más campos en el body:

{ "añoEstreno": 2025 }
DELETE /api/peliculas/:id

Elimina una película específica por su ID.
